Threading & Async I/O

Lec #10 + Notes ▾

Thread Basics

  • Threading allows different parts of a program to run concurrently.
  • Best for I/O-bound tasks (waiting for files/network), not CPU-bound.
import threading

def task(name):
    print(f"Thread {name} running")

t1 = threading.Thread(target=task, args=("T1",))
t2 = threading.Thread(target=task, args=("T2",))
t1.start()
t2.start()
t1.join()   # wait for t1 to finish
t2.join()

Race Condition

Occurs when two or more threads access shared data simultaneously → unpredictable result.

Solution: Lock

lock = threading.Lock()

def withdraw(amount, lock):
    lock.acquire()
    global balance
    balance -= amount
    lock.release()

Async I/O – Event Loop

  • async def func(): → coroutine function
  • await expr → pause and let the event loop run another task
  • asyncio.create_task(coro) → wraps a coroutine into a task
import asyncio

async def task1():
    print("Start task1")
    await asyncio.sleep(1)
    print("End task1")

async def main():
    t1 = asyncio.create_task(task1())
    t2 = asyncio.create_task(task1())
    await t1
    await t2

asyncio.run(main())

Logging

Lec #11 ▾

Why Logging?

  • Tracks events when the program runs.
  • Better than print for complex programs.
  • Can save logs to a file.

Logging Levels (lowest to highest)

LevelNumberUse
DEBUG10Detailed diagnostic info
INFO20General program flow
WARNING30Something unexpected happened
ERROR40A serious problem occurred
CRITICAL50Fatal error – may stop the program

Basic Example

import logging

logging.basicConfig(
    filename='app.log',
    filemode='a',
    format='%(asctime)s - %(levelname)s - %(message)s',
    level=logging.DEBUG
)

logging.debug("debug message")
logging.info("info message")
logging.warning("warning message")
logging.error("error message")

Custom Logger per Module

logger.addHandler(handler)

Flask API

Lec #12 ▾

CRUD with Flask

from flask import Flask, jsonify, request

app = Flask(__name__)
db = [{"id": 1, "name": "Ali"}]

@app.get("/students")
def get_all():
    return jsonify(db)

@app.post("/students")
def add():
    if request.is_json:
        s = request.get_json()
        db.append(s)
        return s, 201
    return {"error": "Must be JSON"}, 415

@app.put("/students")
def update():
    if request.is_json:
        s = request.get_json()
        for i in db:
            if i["id"] == s["id"]:
                db.remove(i)
                break
        db.append(s)
        return s, 201

@app.delete("/students")
def delete():
    if request.is_json:
        s = request.get_json()
        for i in db:
            if i["id"] == s["id"]:
                db.remove(i)
                break
        return s, 201

if __name__ == "__main__":
    app.run(debug=True)
HTTP Status Codes: 200=OK, 201=Created, 415=Unsupported Media Type, 404=Not Found

Generators

Generator (2).pdf ▾

Core Rules

  • Any function containing yield is a Generator.
  • Calling the function returns a Generator Object (does not execute code yet).
  • __next__() executes until the first yield then pauses.
  • When code ends → StopIteration is raised automatically.
  • Can be used directly in a for loop.

Trace Example

def gen(a, b):
    print(a, "before first yield")
    yield 2
    print(b, "before second yield")
    yield 3
    print("last line")

G = gen(1, 2)
for i in G:
    print(i)
Output:
1 before first yield
2
2 before second yield
3
last line

Function Definition & Calls

Lec #2 + Notes ▾

Positional vs Keyword Arguments

def func(a, b, c=2):
    print(a, b, c)

func(1, 2, 5)          # a=1, b=2, c=5
func(1, c=5, b=2)      # a=1, b=2, c=5
func(1, 2)             # a=1, b=2, c=2 (default)
# func(1) → ERROR! b has no default value
A positional argument cannot come after a keyword argument

*args and **kwargs

def func(*args):         # tuple of all positional args
    print(args)

def func2(**kwargs):     # dict of all keyword args
    print(kwargs)

lst = [1, 2, 3]
func(*lst)   # same as func(1, 2, 3)

d = {"b": 20, "c": 10}
def f(a, b, c): print(a, b, c)
f(1, **d)    # same as f(1, b=20, c=10)

def even_numbers(n):
    i = 0
    while i < n:
        if i % 2 == 0:
            yield i
        i += 1

for num in even_numbers(10):
    print(num)

Question 19 – Write Code
Write a thread-safe counter using threading. The increment() function increases the counter by 1. Use a Lock to avoid race conditions. Run two threads each calling increment() 100 times.
✅ Model Answer
import threading

counter = 0
lock = threading.Lock()

def increment():
    global counter
    for _ in range(100):
        lock.acquire()
        counter += 1
        lock.release()

t1 = threading.Thread(target=increment)
t2 = threading.Thread(target=increment)

t1.start()
t2.start()
t1.join()
t2.join()

print("Final counter:", counter)  # should be 200

def count_words(sentence):
    words = sentence.split()
    counts = {}
    for word in words:
        counts[word] = counts.get(word, 0) + 1
    return counts

print(count_words("hi hi hello"))
# {'hi': 2, 'hello': 1}

Question 9 – Write Code
Write a Generator function fibonacci(n) that yields the first n Fibonacci numbers.
Example for n=6: 0, 1, 1, 2, 3, 5
✅ Model Answer
def fibonacci(n):
    a, b = 0, 1
    count = 0
    while count < n:
        yield a
        a, b = b, a + b
        count += 1

for num in fibonacci(6):
    print(num)
# 0, 1, 1, 2, 3, 5
